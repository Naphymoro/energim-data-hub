# SDMX Gate Layer

Defines the SDMX gate used before LEAP/NEMO exports.

Main folders:

- `dsd/`: dataset structure definitions.
- `codelists/`: allowed codes.
- `dataflows/`: dataset-to-dataflow mapping.
- `datasets/`: SDMX-ready dataset outputs.
- `validation_reports/`: gate reports.

The SDMX gate blocks export until metadata, dimensions, provenance, transformation logs, normalized data, and validator quorum are complete.
