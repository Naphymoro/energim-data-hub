# LEAP Normalized Datasets

LEAP-ready files are organized by LEAP module and branch:

- `demand/`
- `transformation/`
- `resources/`
- `socioeconomic/`
- `emissions/`

Each dataset must include provenance, validation status, unit normalization, and an SDMX gate result before export.

`model_input_queue.csv` is the visible staging list showing crawler evidence mapped to LEAP branches and variables. It is not a final LEAP import file.
