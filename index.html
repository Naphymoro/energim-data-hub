Setup Guide

    
Start free with local tracking and downloads. Add Google sync or AI/OCR only when you need those workflows.

  

  

    

      
No credits required

      
Use the app in free local mode first.

      
Local mode keeps the tracker in your browser, exports files to your computer, and lets you paste OCR text manually. You do not need Google Cloud, an LLM key, or paid credits to review the inventory and run the tracker workflow.

    

    

      
      
    

  

  

    
Free local modeBest first run. Browser tracker, Excel export, local downloads, and manual OCR paste. No API credits.

    
Google live syncAdd OAuth only when you want shared Sheets and Drive outputs. GitHub Pages hosting remains free.

    
AI assisted modeAdd an LLM key only when you want automatic mapping or vision OCR. Manual mode still works without it.

  

  

    

      

        
Setup readiness

        
Local mode is ready by default. Optional services appear only as next steps.

      

      

        
        
      

    

    

  

  

    

      
Start free

      
1
Choose Free local mode
Click Use Free Local Mode. This uses browser storage, Excel export, local downloads, and manual OCR paste.

      
2
Use the tracker
Open Tracker, update statuses, assign responsibility, add quality/completeness, and export XLSX when needed.

      
3
Process one sample
Open SDMX Polyglot, paste CSV/table text, identify, validate, transform, and download the outputs locally.

      
4
Handle scanned documents
For a scanned PDF/image, run OCR outside the app first, paste the extracted text, then map it. No API credits are required.

      
5
Deploy free
GitHub Pages serves the static app for free from this repository. No server subscription is needed for the local workflow.

    

    

      
Optional upgrades

      
6
Google live sync
Only needed for shared Google Sheets/Drive. Create OAuth credentials, connect Google, add Sheet/Folder IDs, then initialize the sheet.

      
7
LLM mapping
Only needed for automated AI mapping. Choose OpenAI, Gemini, Anthropic, Qwen, Kimi, OpenRouter, or a custom endpoint, then test it.

      
8
OCR automation
Only needed if you want the app to extract text from images or scanned files. Manual paste remains free.

      
9
Production check
Before official use, run one small dataset from ingest to publish and confirm tracker, outputs, and provenance.

    

  

  
ℹAPI keys and settings are stored in this browser's localStorage only. Data is sent only when you connect Google, call the selected LLM provider, or use a configured OCR endpoint. Google OAuth tokens expire after about 1 hour and may require re-authentication.
