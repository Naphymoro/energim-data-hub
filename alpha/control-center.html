Submit evidence for LEAP/NEMO modelling without using code.

This page helps policy teams register evidence folders for Rwanda energy-system modelling. The ENERGIM data team then processes the request using the controlled workstation pipeline: crawler, OCR, SDMX validation and LEAP/NEMO export preparation.

What the local workstation does

The workstation is operated by the ENERGIM data team. It does not run automatically. It runs only when the team manually starts it.

When started, it creates required folders, writes run logs, checks duplicates, extracts tables, runs OCR when needed, builds SDMX candidates and blocks unvalidated exports.

What this browser page cannot do

A static browser page cannot safely run large crawlers, OCR engines or Python model-export pipelines. This page is the request and communication layer.

No policy maker needs to download JSON, open a terminal, install Python, or run VS Code.
