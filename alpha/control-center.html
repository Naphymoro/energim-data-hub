ENERGIM Alpha Control Center

Policy-maker browser interface. Manual actions only.

Workflow

  1. Create evidence run
  2. Register Google Drive evidence folder
  3. Technical workstation executes OCR and crawler
  4. Review SDMX candidates
  5. Approve validated observations
  6. Generate LEAP and NEMO exports

Operations

Architecture

GitHub Pages provides the interface.

Google Apps Script provides the lightweight backend.

Google Sheets stores run and validation registries.

Google Drive stores evidence and exports.

Windows workstation performs heavy OCR and extraction.
