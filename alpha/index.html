Purpose

ENERGIM Alpha v0.1 is a Rwanda-only modelling-readiness environment that extends the ENERGIM Data Hub philosophy without disrupting the production link.

What Alpha is not

It is not yet an official LEAP model, a NEMO optimizer, or an official NDC reporting system. It prepares the data and governance layer.

Baseline

The baseline anchor is 2015. Historical calibration should cover 2015-2024. NDC 3.0 planning should cover 2025-2035, with 2035-2050 retained for LT-LEDS.

Sources

Tier 1 official sources should control baseline data. EPD and private-sector data should enhance project pipeline and scenario intelligence unless validated by government.

LEAP/NEMO

LEAP mappings should include branches, variables, units, and expressions. NEMO mappings should include technologies, fuels, constraints, costs, capacities, emissions, and time slices.

Beta path

Beta should add persistent storage, Google Sheets integration, OCR services, reviewer workflow, API endpoints, and verified model exports.
